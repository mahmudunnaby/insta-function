Q2: Local Storage Vs Session storage

Although Local Storage & Session storage are alike but there are some key differences between them. The data stored in Local Storage doesn't get expired but in Session storage the data expires once the tab is closed.Local Storage needes to be manually deleted otherwise the data is going to persist in the storage. The session storage data is stored on the basis of protocol of the page.The Session storage created from HTTP request and HTTPS request will deffer even if the page is the same.
